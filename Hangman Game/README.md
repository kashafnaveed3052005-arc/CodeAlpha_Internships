A simple text-based Hangman game built using Python.

## Description
This project is a console-based word guessing game.
The computer randomly selects a word and the player has to guess the word by entering one letter at a time.

The player has 6 incorrect guesses available.

## Features
- Random word selection
- Letter-by-letter guessing
- 6 wrong guess limit
- Win and lose conditions
- Console-based interface

## Concepts Used
- random module
- while loop
- if-else statements
- strings
- lists
- user input

## How to Run

1. Download or clone the repository
2. Open terminal in project folder
3. Run:

python hangman.py
